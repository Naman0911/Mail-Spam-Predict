# Mail Spam Predict

## Overview
Mail Spam Predict is a machine-learning based email classification project that detects whether an email is spam or not spam using structured features from the Spambase dataset.

The project uses AdaBoost as the primary model and achieves strong predictive performance.

## Model Performance
- AdaBoost Accuracy: **93%**

## Dataset
- UCI Spambase Dataset
- Features include word frequency, character frequency, and capitalization statistics.

## Workflow
1. Data loading and preprocessing
2. Feature analysis and visualization
3. Model training using AdaBoost
4. Evaluation using confusion matrix and accuracy score

## Results
The AdaBoost classifier achieved 93% accuracy with a balanced confusion matrix, indicating effective spam detection with low false positives.

- scikit-learn
- pandas
- numpy

- seaborn
